Installation

Choose your platform and get started in seconds

🐧 Linux

curl -L -o sa https://github.com/isathish/SAPythonPackageManager/releases/latest/download/sa-x86_64-unknown-linux-gnu chmod +x sa sudo mv sa /usr/local/bin/

🍎 macOS

# Intel Macs curl -L -o sa https://github.com/isathish/SAPythonPackageManager/releases/latest/download/sa-x86_64-apple-darwin # Apple Silicon (M1/M2/M3) curl -L -o sa https://github.com/isathish/SAPythonPackageManager/releases/latest/download/sa-aarch64-apple-darwin chmod +x sa && sudo mv sa /usr/local/bin/

🪟 Windows

# PowerShell Invoke-WebRequest -Uri "https://github.com/isathish/SAPythonPackageManager/releases/latest/download/sa-x86_64-pc-windows-msvc.exe" -OutFile "sa.exe" # Add to PATH or move to desired location

📦 From Source

git clone https://github.com/isathish/SAPythonPackageManager.git cd SAPythonPackageManager/sa cargo build --release sudo cp target/release/sa /usr/local/bin/

Uninstallation

Remove SA from your system

cargo uninstall sa # Removes the SA binary from your system

Usage Examples

Common workflows made simple and intuitive

📦 Add packages to your project

sa add requests numpy pandas # Automatically creates virtual environment and installs packages

🚀 Run scripts with dependencies

sa run --with matplotlib plot_data.py # Installs matplotlib and runs your script in isolated environment

🗑️ Remove packages cleanly

sa remove --package old-dependency # Clean removal with automatic dependency cleanup

🗑️ Uninstall a package

sa uninstall requests # Uninstalls the specified package from the virtual environment

📋 List installed packages

sa list # Beautiful display of all installed packages with versions

🏗️ Build and publish your project

sa build sa publish # Build wheel/sdist and upload to PyPI seamlessly

ℹ️ Get version and system info

sa version # Shows SA version, system info, and environment details
